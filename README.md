# Web-Alsi-Hijau
Aksi Hijau** adalah website *crowdfunding* dan edukasi lingkungan pionir dari pemuda Tangerang Selatan. Platform kreatif ini menyatukan donasi restorasi mangrove pesisir, kalkulator dampak karbon *real-time*, panduan pilah sampah mandiri, serta fitur konsultasi lingkungan bertenaga AI untuk bersama mewujudkan Tangsel bersih, asri, dan bebas polusi.
